# 暖色生活气息 UI 重设计 — 设计规格

**日期**: 2026-04-29  
**方案**: C — 主题驱动 + 视觉层增强  
**技术栈**: React 19 + Ant Design 6 + ECharts 6 + TypeScript

---

## 一、全局设计系统

### 1.1 配色

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 primary | `#f0835b` | 按钮、选中态、链接、强调 |
| 主色 hover | `#e6704a` | 按钮悬停 |
| 主色浅底 | `#fef3ed` | 选中行/卡片底色 |
| 暖金 accent | `#f5a623` | 辅助强调、结余 |
| 收入绿 success | `#38a169` | 收入数字、收入条 |
| 支出珊瑚 danger | `#e07060` | 支出数字、超支 |
| 页面背景 bg | `#fef9f2 → #fdf5e6` | 全局渐变背景 |
| 卡片背景 | `#fffdf9` | 卡片、表格行 |
| 侧边栏背景 | `rgba(255,248,240,0.92)` | 毛玻璃侧边栏 |
| 文字主色 | `#4a3728` | 标题、正文 |
| 文字辅助 | `#8c7568` | 标签、次要信息 |

### 1.2 全局效果

- **卡片圆角**: 16px（原 8px）
- **卡片阴影**: `0 2px 16px rgba(240,131,91,0.06)`（暖色调投影）
- **卡片 hover 阴影**: `0 4px 24px rgba(240,131,91,0.12)`
- **按钮圆角**: 12px，主按钮 hover 时有柔和发光
- **字体**: 保持系统默认，金额数字 `font-weight: 700`
- **渐变动画**: 页面背景 transition 1s，卡片 hover transition 0.3s

### 1.3 侧边栏

- 背景从 Ant Design dark `#001529` 改为浅暖色毛玻璃
- 选中项: 暖橙浅底 `#fef3ed` + 左侧圆点指示器
- Logo 区: "记账助手" 暖橙色，collapsed 时显示暖橙图标

## 九、实现策略

1. **第一步**: 修改 `App.tsx` 中 Ant Design ConfigProvider 的 theme.token，覆盖全局配色
2. **第二步**: 改造 `AppLayout.tsx` 侧边栏为暖色毛玻璃
3. **第三步**: 添加全局 CSS（`index.css` 增强），定义渐变背景、卡片样式、阴影
4. **第四步**: 逐页微调内联样式，更换颜色值、圆角、阴影
5. **第五步**: 配置 ECharts 暖色主题，应用到所有图表
6. **第六步**: 预算管理页表格→卡片网格重构

**不引入新依赖**，仅通过 Ant Design theme token + CSS + 内联样式调整完成。
